1. What is the difference between a block and an inline tag

Block tags: These elements take up the full width available and start on a new line. div, h1, p, and ul.

2. What is normal flow in HTML?

Block elements like div and p stack vertically, starting on a new line.

Inline elements like span and a sit next to each other horizontally.

3. What does float do? Why do you need to clear the float?

What it does: The float property allows elements to be positioned to the left or right of their container, letting text and inline elements wrap around them.

Clearing the float: You need to clear floats to prevent parent containers from collapsing, ensuring proper layout. This can be done using the clear property or clearfix methods.

4. What do absolute, fixed, and relative do?

Absolute: Positions an element relative to its nearest positioned ancestor (not static). It can overlap other elements.

Fixed: Positions an element relative to the viewport, so it stays in place when scrolling.

Relative: Positions an element relative to its original position in the document flow, allowing for adjustments without changing other elements' layout.

How can you make nice navigation buttons for mobile devices?

Use CSS for styling (e.g., rounded corners, padding).

Use flexbox for layout to make them responsive.

Ensure touch targets are large enough for easy tapping.
