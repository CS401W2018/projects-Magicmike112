why do we need a viewport meta tag

It tells the browser how to scale and size the webpage on different devices

when you select the same tag twice and in one of them you say "flex- direction: column" and the other says "flex direction: row" which one takes priority

When you select the same tag twice in your CSS and apply different flex-direction values, the one that appears last in the stylesheet takes priority
