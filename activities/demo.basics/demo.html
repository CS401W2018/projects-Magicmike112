Difference Between Block Tags and Inline Tags

Block Tags: Start on a new line and take up the full width

Inline Tags: Do not start on a new line and only take up as much width as necessary

Margin, Border, and Padding

Margin: Space outside an element, creating distance from other elements.

Border: A line surrounding the element, separating it from the margin.

Padding: Space inside the element, between the content and the border.

Units of Measurement in CSS

Absolute Units: px, pt, cm, in

Relative Units: %, em, rem, vw, vh

When to Use a Figure Tag

Use the figure tag for self-contained content like images, illustrations, or diagrams that can have a caption figcaption, providing context or explanation.
