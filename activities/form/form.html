The form tag has an "action" attribute. What does it do?

Specifies the URL where the form data will be sent for processing when the form is submitted.

The form tag has a "method" attribute. What does it do?

Specifies the HTTP method the browser should use to send the form data.

POST vs GET:

GET: Sends form data in the URL, visible to the user. Typically used for fetching data.

POST: Sends form data in the request body, not visible to the user. Typically used for submitting data that changes the server's state.
